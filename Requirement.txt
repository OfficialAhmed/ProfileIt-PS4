The tool require ImageMagick in order to be able to convert the images 
please consider downloading and installing it before running my tool 

for WIN 64bit
http://www.imagemagick.org/download/binaries/

(installation):

Follow pictures shown in the link
first picture = while installing 
second picture = after done installing

You need to right-click on (This PC) then choose (properties) on the left panel click (Advanced settings) then (environment variables)
after that (new) Lastly you have to set MAGICK_HOME environment variable to the path of ImageMagick (e.g. C:\Program Files\ImageMagick-6.7.7-Q16)

http://docs.wand-py.org/en/0.3.5/guide/install.html


and you're good to go.
NOTE: these steps only for first time next time you only run the tool ProfileIt
